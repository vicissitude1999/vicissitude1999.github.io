Projects

Needle, a mini PyTorch (Python, C++, CUDA)
I built a PyTorch-like deep learning framework with CPU and GPU backends. It supports essential features including backpropagation and most operators. I also added advanced features such as 3D convolution, implicit layer and subgraph backpropagation.

TwitterOps, a lambda architecture big data project (Java, Scala)
Organized 100GB Twitter dataset into tables and analyzed account behavior. Deployed with AWS EC2 and S3. The batch layer saves data to HDFS with Hadoop serialization and Hive; serving layer extracts information with HBase, Spark, Spark SQL, and speed layer sends data and maintain queues with Kafka.

SimpleDB, a prototype database (Java)
Implemented key components of a database from scratch, including physical and logical data management, operators, query optimization and transaction management.

Research: Multi-level objective optimization (Python, PyTorch)
Improved differential neural architecture search (NAS) methods with class weighting and ensemble learning. Managed distributed large-scale experiments with MLOps tools. Achieved SOTA accuracy on CIFAR, ImageNet.

Research: Change detection and VAE (Python, PyTorch)
Conducted visual stream change detection with multi-level variational autoencoder (MLVAE). Augmented benchmark dataset with different simulations in Blender to improve model's generalizability.
